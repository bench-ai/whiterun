Generates a new image from a text prompt using RealVisXL as the model. Generates more realistic images.

Sampler - Determines patterns used to generate the image. Each sampler outputs a different quality at varying speeds so experimentation is needed.
Steps - How many diffusion steps to run, more steps means more quality but longer time to generate.
Guidance - How closely to follow the prompt.
Seed - Value determines noise seed, leave the same if you want to generate the same image given the same prompt, otherwise we recommend leaving it empty for a random seed.
Prompt Strength - How strongly the prompt should affect the image when being generated.
Safety filter - Disabling this option allows the model to generate NSFW images.
