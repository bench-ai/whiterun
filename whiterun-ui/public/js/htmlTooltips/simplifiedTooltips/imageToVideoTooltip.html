Generate a video using an image as a starting point

Max Frames - How many frames to generate the video with. Larger number means longer video.
Steps - How many diffusion steps to run, more steps means more quality but longer time to generate.
Guidance - How closely the model follows the given prompt when generating the image, lower values mean higher creativity while higher values require more detailed prompts.
Seed - Value determines noise seed, leave the same if you want to generate the same image given the same prompt, leaving it empty, generates a random seed.
