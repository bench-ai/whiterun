Generate a new image from a text prompt using Dall-E 3

Inputs:
Input Prompt - A text object or list
Example Input Prompt - Can take an Image Prompt Operator or Prompt Grouper Operator.

Output - An image file
Example Connections - Can be connected to an Image Display Operator in order to display it. If you want to continue modifying it, you can connect it to an Upscaler Operator or a Image to Image Masking Operator

Style - Style of image to generate. Vivid generates hyper-real and cinematic images while natural produces more natural looking images.
Resolution - Determines the output resolution of the generated image.
