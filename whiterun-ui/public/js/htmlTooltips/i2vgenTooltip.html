Generate a video using an image as a starting point

Inputs:
Input Image - An Image File
Example Input Image - Can take the output of a Text to Image Operator or Image Operator.

Prompt - A text object or list
Example Prompt Input - Can take the output of a Image Prompt Operator or Prompt Grouper Operator.

Output - A video file
Example Connections - Can be connected to a Image Display Operator in order to display it.

Number of Frames - How many frames to generate the video with. Larger number means longer video.
Inference Steps - How many diffusion steps to run, more steps means more quality but longer time to generate.
Guidance Scale - How closely the model follows the given prompt when generating the image, lower values mean higher creativity while higher values require more detailed prompts.
Seed - Value determines noise seed, leave the same if you want to generate the same image given the same prompt, leaving it empty, generates a random seed.
