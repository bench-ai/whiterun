Generates portraits and avatars using another portrait image as a starting point.

Inputs:
Input Image - An Image File
Example Input Image - Can take the output of a Image Operator.
Input Image 2-4 (Optional) - Upload more photos of the person to be customized to improve photo fidelity.
Positive Prompt - A text object or list
Example Input Positive Prompt - Can take the output of an Image Prompt Operator.
Negative Prompt - A text object or list
Example Input Negative Prompt - Can take the output of an Image Prompt Operator.

Output - An image file
Example Connections - Can be connected to a Image Display Operator in order to display it. If you want to continue modifying it, you can connect it to a Upscaler Operator.

Style - Determines the style of the output image
Inference Steps - How many diffusion steps to run, more steps means more quality but longer time to generate.
Style Strength Ratio - How much the selected style should influence the generated image.
Guidance Scale - How closely to follow the prompt.
Seed - Value determines noise seed, leave the same if you want to generate the same image given the same prompt, otherwise we recommend leaving it as 0 for a random seed.
Disable safety filter - Checking this option allows model to generate NSFW images.
