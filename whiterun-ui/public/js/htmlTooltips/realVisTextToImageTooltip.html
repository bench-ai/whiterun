Generates a new image from a text prompt using RealVisXL. Generates more realistic images.

Output - An image file
Example Connections - Can be connected to a Image Display Operator in order to display it. If you want to continue modifying it, you can connect it to a Upscaler Operator.

Sampler - Determines patterns used to generate the image. Each sampler outputs a different quality at varying speeds so experimentation is needed.
Inference Steps - Determines the output resolution of the generated image.
Guidance Scale - How closely to follow the prompt.
Seed - Value determines noise seed, leave the same if you want to generate the same image given the same prompt, otherwise we recommend leaving it as 0 for a random seed.
Disable safety filter - Checking this option allows model to generate NSFW images.
Positive Prompt - Text description for the desired image.
Negative Prompt - Text description things you want the model to not include or generate.
