Takes up to five prompts and groups it into a list of prompts

Input - Up to 5 Text Objects
Example Input - Can be used with several Image Prompt Operators to group it into one list of prompts.

Output - A list of text objects
Example Connections - Can be used as the input as the input prompt for operators such as Image to text, Image to Image, Image to Image Masking, ControlNet Tile Upscaler.
