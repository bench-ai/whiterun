Generate an image using another image as a starting point using RealVisXL. Generates more realistic images.

Inputs:
Input Image - An Image File
Example Input Image - Can take the output of a Text to Image Operator or Image Operator.

Prompt - A text object or list
Example Prompt Input - Can take the output of an Image Prompt Operator or Prompt Grouper Operator.

Output - An image file
Example Connections - Can be connected to an Image Display Operator in order to display it. If you want to continue modifying it, you can connect it to an Upscaler Operator.

Sampler - Determines patterns used to generate the image. Each sampler outputs a different quality at varying speeds so experimentation is needed.
Inference Steps - Determines the output resolution of the generated image.
Guidance Scale - How closely to follow the prompt.
Seed - Value determines noise seed, leave the same if you want to generate the same image given the same prompt, otherwise we recommend leaving it as 0 for a random seed.
Prompt Strength - How strongly the prompt should affect the image when being generated.
Disable safety filter - Checking this option allows model to generate NSFW images.
